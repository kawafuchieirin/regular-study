## 📘 勉強会のルール

### 1. 目的を大切にする

継続して学習習慣を身につけることを最優先にする

完璧を目指さず「まずやる」「続ける」ことを重視

### 2. ゆるく・楽しく参加する

途中参加・途中退席は OK

参加できるタイミングで気軽に参加する

### 3. 発言・質問は自由

初歩的な質問も歓迎

わからないことはその場で遠慮なく聞いてよい

発言しなくても参加だけでもOK

### 4. 互いを尊重する

否定しない

知識量に差があっても気にしない

共有内容やミスを責めない

### 5. 記録は簡単に残す

スレッドにメモや参考資料などを残す

できなくても無理にやらなくてよい

小さな積み重ねを大事にする

### 6. 改善していく

毎回終わりに「よかった点」「改善したい点」を簡単に共有

良いと思ったら次回に取り入れる

続けながら自然にブラッシュアップする

## 勉強会のテーマについて

- かわふちの独断と偏見でテーマを決める
- 参加者のアンケートで決める
